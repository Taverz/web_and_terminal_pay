
### О плагине

Предназначен для того чтобы проводить оплату через Юкассу (Yookassa - WEB) и Сбербанк/Сбер терминал(Sber/Sberbank) POS терминал KozenP12.    
Плагин предназначен для организаций.  


## Features
Чеки:
- Выдача чека
- Чек возврата

Терминал:
- Создание оплаты
- Проверка статуса
- Сверка итогов
- Отмена оплаты
- Возврат

Веб оплата:
- Вызов оплаты
- Чек автоматически
- Получение способов оплаты
- Проверка статуса
- Возврат
